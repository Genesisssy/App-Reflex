# 🌸 Proyecto Web con Reflex

## 📖 Descripción del proyecto

Este proyecto consiste en una página web interactiva desarrollada utilizando el framework Reflex y Python.

La aplicación incluye:
- 🌸 Título principal
- 💕 Texto de bienvenida
- 💖 Botón interactivo
- 🌷 Cambio dinámico de mensajes
- 📊 Contador de clics
- 🎨 Diseño personalizado color rosa

---

# 🛠️ Tecnologías utilizadas

- Python 3.12.9
- Reflex
- Poetry
- Git
- Visual Studio Code

---

# ⚠️ Requisitos antes de ejecutar el proyecto

Antes de ejecutar la aplicación es necesario tener instalado:

- Python 3.12.9
- Poetry
- Git
- Visual Studio Code

---

# 📥 Clonar el repositorio

```bash
git clone LINK_DE_TU_REPOSITORIO
```

---

# 📂 Entrar a la carpeta del proyecto

```bash
cd App-Reflex
```

---

# 📦 Instalar dependencias

```bash
poetry install
```

---

# ▶️ Ejecutar la aplicación

```bash
poetry run reflex run
```

---

# 🌐 Abrir en el navegador

Después de ejecutar el proyecto, abrir:

```plaintext
http://localhost:3000
```
